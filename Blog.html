Question-1: What are the different ways to select an element in the DOM?
The different ways to select elements in the DOM are: getElementById(), querySelectorAll(), document.forms, document.images, document.links, document.body & document.head.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML: Gets or sets HTML content, including tags. innerText: Gets or sets only visible text, ignoring hidden elements textContent: Gets or sets all text, including hidden content, without parsing HTML.
Question-3: What is event delegation in the DOM?
Event delegation is a technique where a parent element handles events for its child elements using event bubbling.
Question-4: What is event bubbling in the DOM?
Event bubbling is a DOM mechanism where an event starts from the target element and propagates upwards through its ancestors until it reaches the root (document).
Question-5: How do you create, add, and remove elements using JavaScript?
GUse document.createElement() to create, appendChild() or insertBefore() to add, and removeChild() or element.remove() to remove elements in JavaScript.
